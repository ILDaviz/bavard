# Installation

Add Bavard to your `pubspec.yaml` dependencies:

```yaml
dependencies:
  bavard: ^0.0.21
```

Then run:

```bash
dart pub get
```

## Requirements

- Dart SDK: `>=3.0.0 <4.0.0`

## Development Dependencies

If you plan to use code generation for typed models, add these dev dependencies:

```yaml
dev_dependencies:
  build_runner: ^2.4.0
```
